<h1>Real Time Bus Tracking</h1>
<h3>Description</h3>
<p>This is a project for testing calls to an external API and programatically manipulating the DOM. You can test it yourself and feel free to help out.</p>
<br>
<h3>Installation</h3>
<p>To run this on your envirorment just clone or download it and open the index.html file.</p>
<br>
<h3>Usage</h3>
<p>This is meant as a simple project for testing API calls and DOM manipulation.</p>
<br>
<h3>Support</h3>
<p>No support is provided with this code.</p>
<br>
<h3>Roadmap</h3>
